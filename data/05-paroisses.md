# ⛪ Paroisses

Chaque paroisse contient une ou plusieurs églises/chapelles et est rattachée à un ensemble paroissial.

## Types de paroisse

| Code | Description |
|------|-------------|
| `territoriale` | Paroisse territoriale classique |
| `personnelle-rite` | Paroisse personnelle liée à un rite (ex: gréco-catholique, maronite) |
| `personnelle-communaute` | Paroisse personnelle liée à une communauté (ex: portugaise, polonaise) |
| `universitaire` | Paroisse/aumônerie universitaire |

## Types de lieu de culte

| Code | Description |
|------|-------------|
| `cathedrale` | Cathédrale |
| `basilique` | Basilique |
| `eglise` | Église paroissiale |
| `chapelle` | Chapelle |
